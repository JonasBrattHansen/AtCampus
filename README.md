# AtCampus 

## Server

### Installing docker
[Download here](https://docs.docker.com/get-docker/)

### Docker setup
**Make sure to have docker running**
```console
cd Server/src/main/resources/docker/
docker compose down
docker compose up -d
```
## Client

### Start the expo app

```console
cd Client
npm start
```
